<a name="_page21_x0.00_y431.00"></a>Si encuentras algún problema al intentar iniciar sesión, aquí te explicamos los errores más comunes y cómo resolverlos:

1. ***Campos vacíos***
- **Mensaje de error**: "Por favor, introduce tu correo electronico" o "Por favor, introduce tu contraseña" .
- **Causa**: Has dejado uno o dos campos sin rellenar.
- **Solución**: Asegúrate de que los campos de **correo electrónico** y **contraseña** estén completo y vuelve a intentar iniciar sesión.

![](Aspose.Words.ff742217-3c2b-47fe-b004-c9b8cdb527af.028.jpeg)

2. ***Correo inválido***
- **Mensaje de error**: "Por favor, introduce un correo electronico válido".
- **Causa**: El formato del correo electrónico es incorrecto (por ejemplo, falta el "@" o el dominio).
- **Solución**: Revisa que el correo esté escrito correctamente.

  Asegúrate de que incluya un "@" y un dominio válido (por ejemplo, "<usuario@ejemplo.com>").

  Intenta nuevamente.

![](Aspose.Words.ff742217-3c2b-47fe-b004-c9b8cdb527af.029.jpeg)

3. ***Usuario no existente***
- **Mensaje de error**: "El correo electrónico no está registrado".
- **Causa**: El correo electrónico que ingresaste no está asociado a ninguna cuenta.
- **Solución**:Verifica que el correo esté bien escrito.

  Si no tienes una cuenta, selecciona "Registrarse" para crear una.

<a name="_page25_x0.00_y221.00"></a>***Glosario de términos japoneses***

Aquí te explicamos algunos términos que puedes encontrar en nuestro menú para que puedas elegir con confianza:

- **Tempura**: Fritura ligera japonesa. Ejemplo: Langostinos en tempura.
- **Nigiri**: Pequeñas porciones de arroz con pescado o marisco encima.  Ejemplo: Nigiri de salmón.
- **Maki**: Rollos de arroz y pescado envueltos en alga nori.  Ejemplo: Maki de atún.
- **Sashimi**: Lonchas finas de pescado crudo servidas sin arroz. Ejemplo: Sashimi de lubina.
- **Wasabi**: Pasta picante verde hecha de rábano picante. Se sirve como acompañamiento.
- **Gari**: Jengibre encurtido que se sirve para limpiar el paladar entre bocados. Se sirve como acompañamiento.

\## Introducción 

Kibo es una aplicación de gestión de usuarios, productos y pedidos. Se ha desarrollado con una arquitectura MVC utilizando \*\*Spring Boot\*\* para la API REST y \*\*IONIC\*\* para la interfaz de usuario. Los usuarios pueden realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre los usuarios, sus teléfonos y los productos. 

\## Tecnologías Utilizadas 

- \*\*Backend\*\*: Spring Boot, JPA, Hibernate, MySQL 
- \*\*Frontend\*\*: Ionic 
- \*\*Base de Datos\*\*: MySQL 
- \*\*Herramientas Adicionales\*\*: Postman para pruebas API

\## Base de Datos 

\### Tablas y Relaciones 

- \*\*USUARIOS\*\* (id\_usuario, NombreCompleto, Correo\_electronico, Teléfonos, Dirección, Contraseña, Tarjeta, Tipo\_usuario) 
- \*\*PEDIDOS\*\* (id\_pedido, id\_usuario, Fecha\_pedido, Estado\_pedido, Tipo\_entrega, Total) 
- \*\*PRODUCTOS\*\* (id\_producto, Nombre\_producto, Precio, Descripción, Alérgenos) 
- \*\*DETALLEPEDIDOS\*\* (id\_detalle, id\_pedido, id\_producto, Cantidad, Subtotal) 
- \*\*TELEFONOS\_USUARIOS\*\* (id\_usuario, telefono) 

\### Relaciones Clave 

- Un usuario puede realizar muchos pedidos (1:N) 
- Un pedido puede contener muchos productos a través de la tabla DETALLEPEDIDOS (1:N) 
- Un usuario puede tener múltiples teléfonos (1:N)

\## API - Endpoints 

\### UsuariosController 

- `GET /kibo/usuarios` - Obtiene todos los usuarios 
- `POST /kibo/usuarios` - Crea un nuevo usuario 
- `PUT /kibo/usuarios/{id}` - Actualiza un usuario 
- `DELETE /kibo/usuarios/{id}` - Elimina un usuario 

\### ProductosController 

- `GET /kibo/productos` - Obtiene todos los productos 
- `POST /kibo/productos` - Crea un nuevo producto 
- `PUT /kibo/productos/{id}` - Actualiza un producto 
- `DELETE /kibo/productos/{id}` - Elimina un producto

\### Dependencias en `pom.xml`

\## Ejecución del Proyecto 

1. Configurar MySQL y crear la base de datos `kibo` 
1. Ejecutar la aplicación con `mvn spring-boot:run` 
1. Acceder a los endpoints a través de Postman o navegador (`http://localhost:8080/kibo/usuarios`)
